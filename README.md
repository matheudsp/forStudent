
# For Student Project

Aplicativo feito durante matéria de programação para dispositivos móveis



## Ferramentas

 - Expo
 - React Native
 - NativeWind


## Executar localmente

Clone o projeto

```bash
  git clone https://github.com/matheudsp/forStudent
```

Vá para o diretório clonado

```bash
  cd forStudent
```

Instalar dependências com yarn

```bash
  yarn install
```

Iniciar o servidor

```bash
  yarn start
```
